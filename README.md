## Requirements

- Python >= 3.6
- NumPy >= 1.13.3
- Scikit-learn >= 0.20
- Cvxpy >= 1.0.25
- Mosek >= 9.1.13

## Running the scripts

To train and test the BCR model in the terminal, use:

```bash
$ python run_itwsvm_dc.py
```
